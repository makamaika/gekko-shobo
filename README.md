# 月光書房 — 十二刻（とき）、百四十四の物語

彗星の見える夜にだけ路地の奥に現れる古書店「月光書房」。その棚から借り受けた、という体裁の **日本語短編集** です。
一日の **十二の刻（十二時辰）** を一巡し、各刻に **12編ずつ、全144編** を収めています。夜明け前の暗がりから、朝、真昼、黄昏、宵、そして真夜中・丑三つ時まで――時刻ごとに異なる情感の物語がならびます。

各編は文庫本でおよそ **24頁以上**（全144編で **約230万字**）。ジャンルも時代も世界もばらばらの物語が、共通の符牒（**ミラ彗星／月光堂の青いマッチ箱／子守唄「眠れ、眠れ、星が海をわたるまで」／現れて消える書店・月光書房／片方の手袋**）によって、ゆるやかにひとつの空の下へ繋がっています。

そのまま **GitHub Pages** で公開できる静的サイトとして組み上げてあります。
**公開URL → https://makamaika.github.io/gekko-shobo/**

## ディレクトリ構成

```
index.html                 … 入口（ポータル：十二刻を一日の順にならべた目次）
about.html                 … 月光書房について
vol1〜vol12/
  index.html               … 各巻の目次
  about.html               … この巻について
  connections.html         … 符牒の早見表
  stories/01.html〜12.html … 各物語（全144編）
assets/style.css           … 夜空／紙テーマ・縦書き切替などのスタイル
assets/app.js              … 文字サイズ・縦書き・テーマの切替（localStorage 保存）
.nojekyll                  … GitHub Pages の Jekyll 処理を無効化
tools/
  build.py                 … サイト生成スクリプト（site.json + data/ → HTML）
  site.json                … 全12巻・各編のメタ情報（時刻・タイトル・著者名など）
  plan.md                  … 制作ロードマップ
  data/_portal.md, _about.md, _connections_default.md … 入口・総序・符牒の共通本文
  data/vol1〜vol12/story-01.md〜12.md, _about.md（各巻の本文と序）
```

※ folder id（vol1〜vol12）は制作順。ポータルでの並びは各巻の `cycle`（一日の流れ）順です。

## 本文を編集して作り直す

本文は `tools/data/<巻>/story-XX.md`、メタ情報は `tools/site.json` を編集し、再生成します。

```bash
python tools/build.py
```

## 読み方の工夫

- 右上のボタンで **紙／夜空テーマ**、**文字サイズ**、物語ページでは **縦書き／横書き** を切り替えられます（設定はブラウザに保存）。
- 各物語の末尾「この物語にひそむ符牒」で、その編が他の編と分かち合っているモチーフが分かります。
- どの刻から読んでも構いません。時刻の違う物語たちが同じ星の下で書かれていることに気づいたとき、二十四時間が、ひとつの大きな夜空になります。
